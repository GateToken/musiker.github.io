关于我们

MUSIKER 是音乐界的IMD，音乐人（MUSIKER）平台通过区块链技术的引入，致力于打造一个连接整个音乐生态的去中心化生态链，形成以区块链为底层技术的交易市场，
有效解决行业痛点，帮助连接超过20万种风格的音乐家和活动组织者更好的发展。

创作者版权库

音乐人（MUSIKER）将搭建一个面向所有人开放的全球音乐版权库。通过区块链来搭建音乐版权库，收录创作者的作品信息，准确记录创作者的创作历史，利用区块链技术进行版权交易追溯，确保作品交易有据可查，有法可依。

作品交易平台

为了满足市场对音乐品的需求，商家可以提出自己的需求，创作者可以提交自己的作品集，两者进行匹配，我们提供这种撮合交易平台，从而更好的繁荣市场发展。

全球自治社群

区块链技术的发展离不开社群的支持，从区块链诞生的那一刻起，就与社群发展模式相伴而生。平台会内化社群自治机制，通过代币的流通和成员的贡献，共同发展MUS社群并且从中获益。

技术特性

共识机制

所谓共识，是指多方参与的节点在预设规则下，通过多个节点交互对某些数据、行为或流程达成一致的过程，
共识机制是指定义共识过程 的算法、协议和规则。

密码学原理

在音乐人（MUSIKER）中，信息的传播按照公钥、私钥这种非对称数字加密技术实现交易双方的互相信任，有用另一个密钥才能解开的过程。

分布式存储

音乐人（MUSIKER）生态系统区块链中的分布式存储，参与的节点各自都有独立的、完整的数据存储，区块链每个节点都按照块链式结构存储完整的数据。

智能合约

智能合约是一种应用，它能保存价值，存储数据，封装代码，执行计算任务合约是代码(它的功能)和数据(它的状态)的集合，存在于区块链的特定地址。

ERC-20代币合约标准

ERC-20 标准是2015年1月份推出的，这种规则具有通用性，任何ERC-20代币都能够激励兼容以太坊钱包（几乎支持所有以太币的钱包）。

公开资产登记

MUS是基于区块链发布的代币，经过智能合约确认之后，所有的数据都公开、透明、不可篡改。不会出现货币增发，虚假资产的情况。

生态结构

本系统中的音乐艺术生态存在5种角色，系统内角色：平台、创作者、消费者、推广者，外部流转角色：资产交易。
通过他们之间的资产流通，实现了音乐人（MUSIKER）生态系统里的资产流通。

撮合交易流程

平台连接创作者和商家（第三方平台），创作者提供音乐资源，商家提供需求，商家支付费用给创作者，各取所需，完成交易，交易会被记录在区块链系统中，开放查询。

发行计划

High Quality Process

MUS Token是基于ETH(ERC-20)生成的代币。发行总量为100万枚，其总量上限已设定，不可更改，不可增发。

初期发售所筹得的资金将用于以下用途：团队开发音乐版权交易平台国际版，整合区块链技术使平台具备数字货币与艺术作品进行交易的能力；包含此过程中所需的技术执行、营运、推广费用。基金会的编制明确、不可篡改。不会出现货币滥发，虚假资产的情况，确保公平。基金会将会对项目方和团队持有的代币实施逐步释放的方案。

应用流通

MUS Token 作为音乐人（Musiker）生态系统中唯一的价值通证，因其本身的稀缺性（100万枚）和流通价值，拥有巨大的升值空间。
其可在以下几个场景中获得无障碍流通。

  • 广告

  • 对于持有 MUS Token 的音乐家而言，可以通过代币在音乐人（Musiker）生态系统以及接入的第三方音乐平台进行广告投放，以促进音乐作品的传播、销售，同时提升自身的知名度和影响力。

  • 影音

  • 持有 MUS Token 可以享受更多影音消费带来的乐趣，未来，随着 MUS Token 的发展，用户可以享受在全球范围内影片、音乐、游戏等的购买。同时，平台生态内的相关功能也将可以通过相关DAPP实现体验。

  • 娱乐

  • 游戏、娱乐等均可通过 MMUS Token 进行支付，与全球爱好者一起感受数字货币的魅力。未来，随着DAPP应用的发展，各类垂直DAPP将和现在的app服务一样，但具备区块链技术作为支撑，更加安全、稳定、快捷。

  • 购买

  • 消费性支付可以通过 MUS Token 实现，摆脱现有跨境支付体系的限制，做到点对点、即时到账。持有者在线上消费更加便捷。借助区块链全程可追溯、无法篡改、抵赖的特性，让全流程环节更加流畅。
